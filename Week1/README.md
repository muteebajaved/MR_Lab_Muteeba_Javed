# Mobile Robots Lab – Week 1

This lab focused on Linux basics, ROS 2 concepts, workspace setup, and creating the first ROS 2 Python node.

## Commands Used
- ros2 --version
- colcon build
- ros2 pkg create
- ros2 run my_first_pkg simple_node

## Problems Faced
Initially ROS 2 commands were not recognized due to missing sourcing.
This was solved by sourcing the ROS 2 and workspace setup files.

## Reflection
This lab helped me understand the basic ROS 2 workflow.
I learned how workspaces and packages are structured.
Creating and running my first node improved my confidence.
Understanding sourcing and colcon build was very important.
This lab built a strong foundation for future ROS development.
