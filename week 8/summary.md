I designed a custom mobile robot using URDF in ROS 2. I started with a basic robot base and then extended the design by adding multiple components.

The following customizations were made:

Added two wheels using cylindrical geometry for differential drive movement.
Added a camera sensor on top of the robot for future vision applications.
Added a LiDAR sensor using spherical geometry for environmental sensing.
Added an antenna structure for decorative and structural enhancement.
Used different geometric shapes such as box, cylinder, and sphere.
Added different joint types including fixed joints and continuous joints.
Visualized the robot successfully in RViz and verified the frame connections using TF tools.
